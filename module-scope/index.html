module-scope

üìç Get __dirname and __filename in native harmony modules

When running native harmony modules (node --experimental-modules myapp) the classic module scope signature is not available (exports, require, module, __filename, __dirname). This module helps you get __dirname and __filename values easily from import.meta.url:

import scope from 'module-scope';

const {__dirname, __filename} = scope(import.meta.url);
